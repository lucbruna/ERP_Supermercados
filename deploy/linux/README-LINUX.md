# Instalação Linux

## Requisitos
- Ubuntu 20.04+ / Debian 11+ / CentOS 8+ / Fedora 36+
- 4GB RAM mínimo
- 20GB disco
- Acesso root

## Instalação Rápida
```bash
sudo bash install.sh
```

## Pós-instalação
```bash
sudo systemctl start crm-supermercado
sudo systemctl enable crm-supermercado
```

## Acesso
- Dashboard: http://SEU_IP:3081
- PDV: http://SEU_IP:3080
