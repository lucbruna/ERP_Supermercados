# Icones do Instalador CRM Supermercado

Substitua estes icones pelos arquivos .ico finais antes de compilar o instalador:

| Arquivo           | Tamanho | Finalidade                         |
|-------------------|---------|------------------------------------|
| icon.ico          | 256x256 | Icone do instalador e do programa  |
| dashboard.ico     | 64x64   | Atalho do Dashboard no menu iniciar|
| pdv.ico           | 64x64   | Atalho do PDV no menu iniciar      |

### Ferramentas recomendadas:
- **RealWorld Icon Editor** (gratuito): https://www.rw-designer.com/icon-editor
- **GIMP** com plugin .ico: https://www.gimp.org
- **IconArchive**: https://iconarchive.com (icones gratuitos)
